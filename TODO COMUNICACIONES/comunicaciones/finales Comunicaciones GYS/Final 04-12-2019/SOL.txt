Demostración:
N = α*B (1) siendo α la densidad espectral de ruido.
C = B * log2 (1 + S/N) (2)
lim x->0 de (1+x)^(1/x) = e (3)

1. Reemplazando (1) en (2)
C = B * log2 (1 + S/α*B)
2. Multiplicar por α/α * S/S
C = B * log2 (1 + S/α*B) * α/α * S/S
C = B*α/S * log2 (1 + S/α*B) * S/α
3. Considerando x = S/α*B por lo que 1/x = α*B/S
C = 1/x * log2 (1 + x) * S/α
4. Aplicando propiedad de log
C = log2 (1 + x)^(1/x) * S/α
5. Teniendo en cuenta (3) y que si B tiende al infiinito, x = S/(α*B) tiende a 0.
C = lim x→0 de (log2 (1 + x)^(1/x) * S/α) = log2(e) * S/α = 1.44 S/α
C = 1.44 S/α
